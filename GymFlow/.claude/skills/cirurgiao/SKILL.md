# Cirurgiao de Codigo

## Quando Usar
- Em qualquer pedido de alteracao, correcao de bug, refatoracao ou adicao de features em arquivos existentes.

## Diretrizes Estritas
- PROIBIDO reescrever o arquivo inteiro.
- Forneca apenas blocos de codigo isolados ou diffs cirurgicos das linhas afetadas.
- Substitua partes nao alteradas por comentarios como // ... resto do codigo permanece igual ...
- Sem explicacoes teoricas do porque o codigo funciona, a menos que seja explicitamente perguntado.
